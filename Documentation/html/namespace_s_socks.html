A function to determine which sockets have incoming information waiting.

The SSocks::select() function accepts a vector of pointers to sockets of a uniform type (that is, all of them are the same type of socket, whether TCPSocket, TCPServer, or UDPSocket). It returns a vector of the same type which contains only the members of the input vector which were ready to read or accept from. The use of this function is to wait on a group of sockets until one or more of them has incoming information to be processed. This function blocks for the amount of time specified by 'timeoutSeconds' but will return immediately as soon as one or more sockets indicates that it is ready.

Parameters
socketsA vector of pointers to the sockets you want to check
timeoutSecondsThe maximum number of seconds to wait before giving up.
Returns
A containing only those sockets from the input vector which were ready to read or accept from.
