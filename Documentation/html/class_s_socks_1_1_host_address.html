Detailed Description

Class used to represent an IPv4 internet address and port number.

It has conversion functions that allow it to conver to and from sockaddr and sockaddr_in. Note that SimpleSocks is presently IPv4 only, so this class - especially - would need a rewrite to support IPv6.
